# Design

## Current Prototype Observations

The static prototype uses a premium editorial design language with:

- dark photographic hero sections
- burgundy primary actions
- gold accent rules
- cream paper backgrounds
- serif headlines
- compact uppercase navigation and labels
- service cards with top accent rules
- calm, high-contrast page structure

Preserve the brand feel while updating positioning from revenue-leak assessment language to AI Operations Blueprint language.

## Site Direction

The new app uses:

- Next.js app router
- TypeScript
- component-based layout
- CSS custom properties
- minimal dependencies
- copied static assets under `public/`

## Accessibility

- Use semantic landmarks and headings.
- Use descriptive link and button text.
- Keep strong color contrast.
- Ensure required fields are visible and server-validated.
- Avoid hover-only communication.
